Win themes should have been developed during the capture stage by the sales lead, accompanied by a win strategy. A separate bid product (in development) manages this process.

**Why:** At ITT commencement, win themes are being fine-tuned based on actual documentation — not created from scratch. Activities SAL-04, 05, 06 cover this refinement. Win themes clearly differentiate a supplier's value proposition and define the outcome the proposal delivers.

**How to apply:** Design win theme management in the bid execution product as an import/refine workflow, not a creation workflow. The starting point is themes that already exist from capture. Provide framing context for AI in how responses should be written. Win themes are written statements — they need to be substantive, not just labels.
