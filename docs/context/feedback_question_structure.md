---
name: ITT question structure — one question, one owner
description: ITT questions with sub-parts (Q1, 1.1, 1.2) are one production unit with one owner, one mark allocation
type: feedback
---

A quality question like Q1 with sub-parts 1.1, 1.2, 1.3 is ONE ResponseSection and ONE ResponseItem. The sub-questions are traffic-light signalling for answer structure, not separate production items. One owner, one set of marks, one tracking unit.

**Why:** This is how procurement works. The available marks are singular based on the question's weighting. A single bid manager owns the response and may call on others to support, but tracking is one person.

**How to apply:** ResponseSection-to-ResponseItem is always 1:1. Never model sub-questions as separate ResponseSections. The sub-question structure lives within the questionText field of ResponseSection.
