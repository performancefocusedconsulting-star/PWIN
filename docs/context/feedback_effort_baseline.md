Industry baseline for qualitative bid questions: 2 person-days of effort per page of text (storyboard through to red-review-ready draft, excluding images). At ~500 words/page this means a 3,000-word question = 12 person-days at standard complexity.

**Why:** User has direct experience as a bid manager and provided this as a proven calibration metric. AI-assisted drafting is expected to reduce this but the baseline should reflect traditional effort.

**How to apply:** Use this as the "high" complexity default in effort derivation calculations. Lower complexity tiers (low/medium) reflect AI-assisted acceleration. Always present effort as a suggestion the bid manager can override.
