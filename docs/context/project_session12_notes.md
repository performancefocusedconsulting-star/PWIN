- SAL-02 L2.3 added: transformational disruption assessment — can we leapfrog incumbent through innovation? Tandem test: disruption potential AND evaluation framework alignment
- SAL-02 synthesis moved from L2.2 to L2.3 — now pulls from all 9 upstream artefacts across 3 L2s
- SAL-04 L2.2 is "Refinement & Messaging" not "Value Proposition" — messaging informs solution design, commercial proposition, and storyboard drafting
- Win themes originate from capture (Win Strategy product) or developed cold-start — SAL-04 handles both modes
- Elevator pitch / value statement dropped from SAL-04 — those are storyboard artefacts (BM-10)
