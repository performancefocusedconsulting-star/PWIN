## Action 1: Design incumbent vs challenger question variants

The 24 Qualify questions are currently framed from a challenger perspective. When the user selects "We are the incumbent" in the pursuit context, the questions should adapt:

- Same 6-category model and scoring structure
- Question text, rubrics, and inflation signals change based on position
- Incumbent questions probe: complacency risk, relationship decay, transformation investment, re-compete readiness, client switching signals
- Challenger questions probe: champion development, incumbent unseating strategy, competitive differentiation, transition credibility

Recommended approach: Option B — one set of questions with variant framing per position, not two separate sets. Keeps the architecture clean.

This is a design session — needs user input on the framing of each variant before building.
